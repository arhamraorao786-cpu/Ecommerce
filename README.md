# Django E-Commerce Store

A beginner-friendly e-commerce web application built with Django as part of my journey learning Django fundamentals. This project covers core e-commerce workflows including product browsing, cart management, and order tracking.

## 🚀 Features

### 🛍️ Product Management
Admin/users can add products, each organized under specific **categories** for easy browsing and navigation.

### 🗂️ Category-Wise Product Display
Products are displayed **grouped by category**, allowing users to browse and find items relevant to their interest quickly.

### 🛒 Cart Management
- Users can **add products to cart**.
- Users can **remove products from cart** at any time before checkout.

### 📦 Order Placement
Users can **place an order** directly from their cart, converting selected items into a confirmed order.

### 📍 Order Tracking
Users can **track the status of their placed orders**, keeping visibility into where their order stands after purchase.
